# Whack

A beginner-friendly Unity starter for a simple "slap the face" game.

This repo gives you:

- A clean Unity folder layout
- Starter C# scripts for slap gameplay
- A simple avatar-loading path that can read a local image file
- Setup notes so you can open the project in Unity and wire it together

Start with [docs/UNITY_SETUP.md](/Users/wru/DEV/Projects/GitHub/whack/docs/UNITY_SETUP.md).
